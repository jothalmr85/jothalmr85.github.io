Course Descriptions

CSC-3820: Penetration Testing
This course focuses on ethical hacking techniques used to identify, exploit, and remediate security vulnerabilities in systems and networks. Emphasis is placed on reconnaissance, scanning, exploitation, and reporting in a controlled and authorized environment.

MTH-3010: Cryptography
This course explores the mathematical foundations of cryptography, including classical and modern encryption techniques. Topics include symmetric and asymmetric encryption, hash functions, digital signatures, and secure communication protocols.

CSC-4210: Operating Systems
This course examines the design and implementation of operating systems. Key topics include process scheduling, memory management, file systems, concurrency, and synchronization, with an emphasis on understanding how software interacts with hardware.

CSC-3212: Web Technologies
This course introduces modern web development techniques, including HTML, CSS, and JavaScript. Emphasis is placed on responsive design, accessibility, usability, and building websites that provide meaningful user experiences.
